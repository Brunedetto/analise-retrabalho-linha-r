# analise-retrabalho-linha-r
# 📊 Análise de Retrabalho – Linha R

## 📌 Objetivo
Projeto de análise de dados inspirado em contexto real de linha produtiva, com foco na mensuração e simulação do impacto financeiro de retrabalho.

---

## 🔎 Diagnóstico Operacional
- 9.823 peças retrabalhadas no mês  
- Impacto financeiro aproximado de R$ 4.911,50  
- Taxa média de retrabalho de 3,95%  
- Identificação de evento crítico com impacto relevante  

---

## 📈 Simulação de Melhoria Operacional
Implementação de parâmetro dinâmico para simulação de redução na taxa média de retrabalho.

- Redução de até 2 p.p.
- Economia potencial de até R$ 1.870/mês
- Redução estimada de 38% do impacto mensal

---

## 🛠️ Tecnologias Utilizadas
- Python (pandas)
- Power BI
- DAX
